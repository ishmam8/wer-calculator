# WER Calculator

A web application that calculates Word Error Rate (WER) between a reference and hypothesis sentence, with animated alignment visualization.

## How to Run
npm install
npm run dev

## Color Legend
- 🟢 Green — Match
- 🟠 Orange — Substitution
- 🔴 Red — Deletion
- 🔵 Blue — Insertion
